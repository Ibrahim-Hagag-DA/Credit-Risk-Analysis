# Credit Risk Analysis Dashboard

##  Project Overview
This project presents an interactive Power BI dashboard that analyzes customer credit risk using demographic, financial, and loan-related data. The dashboard helps identify default patterns, evaluate loan performance, and support data-driven lending decisions.
## Tools Used
- Power BI
- Microsoft Excel

  ##  Objectives
- Analyze customer credit risk data.
- Identify high-risk and low-risk customers.
- Visualize key risk indicators using Power BI.
- Support better decision-making through interactive dashboards.

-  ## Dataset
The dataset contains customer demographic, financial, and loan-related information used to analyze credit risk.
It includes:
- Customer information (Age, Gender, Marital Status, Education)
- Financial information (Income, Other Debt, Debt-to-Income Ratio)
- Loan details (Loan Amount, Interest Rate, Loan Grade, Loan Term)
- Credit history (Past Delinquencies, Previous Default, Credit Utilization Ratio)

  ## Dashboard Highlights
- Total number of loan applicants
- Default rate analysis
- Average interest rate
- Average loan amount
- Average debt-to-income ratio
- Default rate by loan grade
- Repaid vs Defaulted loans
- Prior default history analysis
- Default rate by loan purpose
- Default rate by home ownership
- Average loan amount by loan grade
- Interactive filters for Country and Loan Grade

##  Key Insights
- The overall loan default rate is **22%**.
- Applicants with **Loan Grade G** have the highest default rate (98%).
- Customers with a previous default history are more likely to default again.
- Debt Consolidation loans show the highest default rate among loan purposes.
- Higher loan grades are associated with larger average loan amounts.
